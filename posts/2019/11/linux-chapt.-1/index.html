Istilah-istilah

UNIX = sistem operasi pertama yang ditulis dengan bahasa C

LINUX = inux adalah sistem operasi yang berasal dari pengembangan varian Unix

**GPL** = GPL adalah Lisensi dari perangkat lunak bebas tersebut.

GNU = suatu sistem perangkat lunak komputer (Software) sepenuhnya terdiridari perangkat lunak bebas (Freeware)

KERNEL =inti dari sebuah OS yang menjembatani antara hardware dengan software. kernel tidak berinteraksi langsung dengan pengguna, tapi cenderung berinteraksi dengan shell atau program” lain. linux sendiri menggunakan kernel GNU/LINUX

CLI = COMMAND LINE INTERFACE

GUI = GRAPHICAL USER INTERFACE

SHELL = sebuah program (CLI) yang berinteraksi langsung dengan user

DISTRO = varian dari linux

nano, vim, gedit = merupakan text editor/notepad yang ada pada linux

VIRTUAL MACHINE = sebuah mesin virtual untuk menjalankan os

REPOSITORI = kumpulan software khusus untuk Linux yang kita gunakan, dan hanya tersedia di server.

Command dasar

perintah dasar yang akan sering di pakai di linux. jadi kamu bakal sering menggunakan keyboard dan semua perintah harus di ketik seperti membuat folder, meng-copy file, menghapus file dan bahkan menginstall aplikasi menggunakan perintah yang harus di ketik.

cd *(Change Direktori)*

berbindah Direktori ke Downloads

$ cd /home/Downloads/

direktori yang dimaksud disini ialah folder

pwd (Print Working Direktori)

$ pwd

ls (List)

$ ls

menampilkan isi yang ada pada direktori saat ini

mkdir (Make Directory)

$ mkdir belajar

membuat direktori dengan nama belajar

touch

$ touch isi.txt

membuat file dengan nama isi.txt

cp (Copy)

$ cp isi.txt /home/

meng copy file*isi.txt*ke direktori*home*

mv (Move)

$ mv isi.txt /home/

memindahkan file*isi.txt*ke direktori*home*

rm (Remove)

$ rm isi.txt

menghapus file*isi.txt*

rm -rf (Remove direktori beserta isinya)

$ rm -rf /belajar/

menghapus direktori belajar dan isinya
